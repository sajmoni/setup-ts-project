# setup-ts-project

> Setup everything you need for a TypeScript project

## :sparkles: Features

 - `npm-init-ex`
 - `add-prettier`
 - `add-typescript`
 - Adds a simple readme
 - Adds test (maybe...)
 - Adds eslint (maybe...)
 - Creates initial commit
 
## Usage

First create a repository on Github and clone it

Then run:

```sh
npx setup-ts-project@latest
```
